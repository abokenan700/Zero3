# Blinkit Home Page Visual Analysis

## Screenshot synthesis

The three screenshots describe one Blinkit-style home page. The full-length capture defines the full vertical order. The default viewport clarifies the hero/header proportions. The scrolled capture clarifies that the search bar and category tabs become a sticky cream-colored header while the delivery/address block scrolls away.

## Page structure

1. Android status bar over a gold delivery header.
2. Delivery header with `Blinkit in`, a large `8 minutes` delivery time, a rounded `24/7` badge, `Chandigarh, India` with a dropdown chevron, wallet button, and profile button.
3. Rounded white search input with search icon, placeholder text, right divider, and microphone icon.
4. Horizontal category tabs: All, Vacations, Electronics, Beauty, Pharmacy, plus a partially visible trailing tab. `All` is active and uses a black underline.
5. Gold welcome banner with shopping bags and copy `WELCOME` and `Order now & enjoy FREE delivery`.
6. Bestseller mosaic cards in a three-column grid.
7. Category grids with four columns: Grocery & Kitchen, Snacks & Drinks, Beauty & Personal Care, Household Essentials, Stores in spotlight, and Picks for your lifestyle.
8. Horizontal product carousels with product cards, add buttons, prices, quantities, heart icons, timers, and centered `See all products` pills.
9. Featured banner strip.
10. Fixed bottom navigation: Home, Order Again, Categories, Print. Home is active and uses a yellow filled icon plus a black top indicator.

## Visual language

- Primary colors: gold gradient header, white page background, cream sticky header, pale mint category tiles, pale gray bestseller/product containers, green ADD buttons.
- Typography: heavy rounded sans-serif appearance, with extra-bold section titles and delivery time. Body labels use compact bold weights.
- Radii: search 18px, category tiles 16px, cards 12-18px, round wallet/profile controls.
- Shadows: subtle search shadow, sticky header shadow, bottom navigation elevation.
- Scroll behavior: delivery/header scrolls away; search/tabs remain sticky at top with cream background.
